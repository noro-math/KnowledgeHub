1 Linear Algebra Exercises

1.1 Chapter 1: Vector Spaces

1.1.1 Exercise 1.1

Prove that \(-(-v) = v\) for every \(v\) in a vector space \(V\).

TipSolution

Let \(v \in V\). We know that \(-v\) is the additive inverse of \(v\), so \(v + (-v) = 0\).

Since \(-v \in V\), it has an additive inverse, which we denote as \(-(-v)\). By definition: \((-v) + (-(-v)) = 0\).

Now, consider: \[v + (-v) = 0 = (-v) + (-(-v))\]

Adding \(-(-v)\) to both sides: \[-(-v) + v + (-v) = -(-v) + (-v) + (-(-v))\] \[-(-v) + 0 = -(-v) + 0\] \[-(-v) = -(-v)\]

This is always true. Let’s try a different approach.

From \(v + (-v) = 0\), we can add \(-(-v)\) to both sides: \[-(-v) + v + (-v) = -(-v) + 0\] \[(-(-v) + v) + (-v) = -(-v)\]

Since \((-v) + (-(-v)) = 0\), we have \(-(-v) = -(-v)\).

Actually, let’s use uniqueness of additive inverse: Since \(v + (-v) = 0\) and \((-v) + (-(-v)) = 0\), and the additive inverse is unique, we must have \(v = -(-v)\).

1.1.2 Exercise 1.2

Suppose \(v_1, v_2, v_3, v_4\) spans \(V\). Prove that the list \(v_1 - v_2, v_2 - v_3, v_3 - v_4, v_4\) also spans \(V\).

TipSolution

We need to show that every vector in \(V\) can be written as a linear combination of \(v_1 - v_2, v_2 - v_3, v_3 - v_4, v_4\).

Since \(v_1, v_2, v_3, v_4\) spans \(V\), any \(v \in V\) can be written as: \[v = a_1v_1 + a_2v_2 + a_3v_3 + a_4v_4\]

We need to express \(v_1, v_2, v_3\) in terms of our new list: - \(v_4 = v_4\) - \(v_3 = (v_3 - v_4) + v_4\) - \(v_2 = (v_2 - v_3) + v_3 = (v_2 - v_3) + (v_3 - v_4) + v_4\) - \(v_1 = (v_1 - v_2) + v_2 = (v_1 - v_2) + (v_2 - v_3) + (v_3 - v_4) + v_4\)

Therefore: \[v = a_1[(v_1 - v_2) + (v_2 - v_3) + (v_3 - v_4) + v_4] + a_2[(v_2 - v_3) + (v_3 - v_4) + v_4] + a_3[(v_3 - v_4) + v_4] + a_4v_4\]

Collecting terms: \[v = a_1(v_1 - v_2) + (a_1 + a_2)(v_2 - v_3) + (a_1 + a_2 + a_3)(v_3 - v_4) + (a_1 + a_2 + a_3 + a_4)v_4\]

This shows that \(v\) is in the span of \(v_1 - v_2, v_2 - v_3, v_3 - v_4, v_4\).

1.2 Chapter 2: Finite-Dimensional Vector Spaces

1.2.1 Exercise 2.1

Find a basis for the vector space of polynomials in \(\mathcal{P}_3(\mathbb{R})\) that have the property that \(p(2) = 0\).

TipSolution

We need polynomials \(p(x) = a_0 + a_1x + a_2x^2 + a_3x^3\) such that \(p(2) = 0\).

This gives us: \(a_0 + 2a_1 + 4a_2 + 8a_3 = 0\), so \(a_0 = -2a_1 - 4a_2 - 8a_3\).

Therefore: \(p(x) = (-2a_1 - 4a_2 - 8a_3) + a_1x + a_2x^2 + a_3x^3\) \(= a_1(x - 2) + a_2(x^2 - 4) + a_3(x^3 - 8)\)

A basis is: \(\{x - 2, x^2 - 4, x^3 - 8\}\)

We can verify: - These polynomials are linearly independent (check by setting linear combination to zero) - They span the space (any polynomial with \(p(2) = 0\) can be written in this form) - The dimension is 3 (which makes sense: we have one constraint on a 4-dimensional space)

1.2.2 Exercise 2.2

Prove that if \(U_1, U_2, W\) are subspaces of \(V\) such that \(V = U_1 \oplus W = U_2 \oplus W\), then \(U_1 = U_2\).

TipSolution

We need to show \(U_1 \subseteq U_2\) and \(U_2 \subseteq U_1\).

Let \(u_1 \in U_1\). Since \(V = U_2 \oplus W\), we can write: \[u_1 = u_2 + w\] for unique \(u_2 \in U_2\) and \(w \in W\).

Since \(V = U_1 \oplus W\), we also have: \[u_1 = u_1 + 0\] where \(u_1 \in U_1\) and \(0 \in W\).

By uniqueness of the direct sum decomposition \(V = U_1 \oplus W\): - The \(U_1\) component is \(u_1\) - The \(W\) component is \(0\)

But we also wrote \(u_1 = u_2 + w\) where \(u_2 \in U_2\) and \(w \in W\). Since \(u_1 \in U_1\) and \(V = U_1 \oplus W\), we must have \(w = 0\) and the \(U_1\) component equals \(u_2\).

Therefore \(u_1 = u_2 \in U_2\), so \(U_1 \subseteq U_2\).

By symmetry, \(U_2 \subseteq U_1\), so \(U_1 = U_2\).

1.3 Chapter 3: Linear Maps

1.3.1 Exercise 3.1

Give an example of a linear map \(T: \mathbb{R}^4 \to \mathbb{R}^2\) such that \(\dim(\text{null } T) = 2\).

TipSolution

By the fundamental theorem of linear maps: \[\dim \mathbb{R}^4 = \dim(\text{null } T) + \dim(\text{range } T)\] \[4 = 2 + \dim(\text{range } T)\]

So \(\dim(\text{range } T) = 2\).

Define \(T: \mathbb{R}^4 \to \mathbb{R}^2\) by: \[T(x_1, x_2, x_3, x_4) = (x_1 + x_2, x_3 + x_4)\]

Then: - \(\text{null } T = \{(x_1, x_2, x_3, x_4) : x_1 + x_2 = 0 \text{ and } x_3 + x_4 = 0\}\) - \(= \{(a, -a, b, -b) : a, b \in \mathbb{R}\}\) - \(= \text{span}((1, -1, 0, 0), (0, 0, 1, -1))\)

So \(\dim(\text{null } T) = 2\) as required.
