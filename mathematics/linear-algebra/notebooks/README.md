# Linear Algebra Jupyter Notebooks

This folder contains interactive Jupyter notebooks for learning linear algebra concepts through computational examples and visualizations.

## Notebooks

1. **01-vector-spaces.ipynb** - Introduction to vector spaces, operations, and axioms
2. **02-linear-transformations.ipynb** - Linear transformations, matrices, and geometric interpretations
3. **03-eigenvalues-eigenvectors.ipynb** - Eigenvalues, eigenvectors, and applications like PCA

## Getting Started

### Prerequisites
- Python 3.7+
- Jupyter Notebook or JupyterLab
- Required packages: `numpy`, `matplotlib`, `scikit-learn`

### Installation
```bash
pip install jupyter numpy matplotlib scikit-learn
```

### Running the Notebooks
```bash
jupyter notebook
```
Then navigate to this folder and open any notebook.

## Features

- **Interactive Visualizations**: Each notebook includes plots and animations to illustrate concepts
- **Computational Examples**: Hands-on coding examples using NumPy
- **Exercises**: Practice problems to reinforce learning
- **Real-world Applications**: Examples like PCA for data analysis

## Learning Path

1. Start with vector spaces to understand the foundation
2. Move to linear transformations to see how matrices work
3. Explore eigenvalues and eigenvectors for advanced applications
